Tip for the assembly part: you can generate the assembly files by going into project Properties (right-click on project)->Build->C5500 Cumpiler->Advanced Options->Assembler Options and check the "Keep the generated assembly language (.asm) file" (first option). but firstly compile in Release instead of Debug. Then go look at the generated assembly file in the "Release" folder of your project. Have fun copying from the most efficient assembly code derived from your c implementation.
